# 支付对接（v2）

- 说明：仅对接，不自研。封装微信/支付宝支付能力，支持预下单、支付回调、退款、对账下载。
- 接口（内部服务给业务调用，均 POST）：
  - `/api/pay/unifiedOrder` 预下单（返回支付参数/二维码）
  - `/api/pay/refund` 退款申请
  - `/api/pay/close` 关闭订单
  - `/api/pay/reconcile/pull` 拉取对账单
- 回调：`/api/pay/callback/{channel}`（签名校验，幂等处理）
- 关联模块：订单、钱包、对账
- 安全：平台证书/密钥隔离，回调 IP 白名单，重放保护
