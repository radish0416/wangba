# 认证与权限（Auth/RBAC）

- 说明：后台账号登录、角色、权限点（菜单/按钮/API），门店级数据隔离。
- 接口前缀：`/api/auth`
- 接口：`/login` `/logout` `/refreshToken` `/role/create|update|delete|listPage|listAll` `/perm/listAll`
- 查询字段：账户名、角色、状态、创建时间范围。
- 数据库：`db.sql`（见同目录），表：`sys_user` `sys_role` `sys_role_perm` `sys_perm`。
- 流程图：`认证与权限-流程.svg`（占位）。
